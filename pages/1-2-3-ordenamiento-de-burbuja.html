1.2.3 Ordenamiento de burbuja

El algoritmo de ordenamiento de burbuja se inspira en el fenómeno de las burbujas que se forman en un dispensador de agua al abrir la válvula, donde la diferencia de presión permite que el líquido fluya y el aire se introduzca, generando burbujas que ascienden debido a su menor densidad. El proceso de ordenamiento comienza considerando el primer elemento como el pivote, el cual se utiliza para comparar con los elementos que le siguen. Si un elemento subsecuente es menor que el pivote, se intercambian sus posiciones. Al finalizar las comparaciones, el pivote contendrá el elemento más pequeño. Este procedimiento se repite, avanzando el pivote una posición, hasta que el arreglo esté completamente ordenado.
